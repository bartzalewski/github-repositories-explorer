# Github repositories explorer

Link to the app: https://github-repositories-explorer.herokuapp.com/

## What kind of technologies did I use?

- React.js
- Redux
- styled-components
- uuid

## How to run?

1. `cd github-repositories-explorer`
2. `npm install`
3. `npm run build`
4. `npm start`
